# 📖 Overview

Nexus Core is a modern, premium, fully responsive cyber-inspired e-commerce platform designed for high-performance hardware shopping.

It delivers a fast, immersive shopping experience with real-time state management, persistent carts, wishlists, Firebase authentication, and elegant UI animations powered by Svelte 5.

# 🚀 Getting Started

## 1. Clone the repository

```bash
git clone https://github.com/your-username/nexus-core.git
```

## 2. Navigate into the project

```bash
cd nexus-core
```

## 3. Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

---

# 🔥 Firebase Setup

Create a `.env` file in the project root and add your Firebase credentials.

Example:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

# ▶️ Run the Project

Start the development server:
